# Simple Unity Procedural Glitch

A lightweight script that creates a 3D glitch effect by spawning random-colored cubes. 

### How to use:
1. Create a Cube in Unity, make it a **Prefab**, and remove it from the scene.
2. Attach the `GlitchEffect.cs` script to any object (e.g., your enemy or player).
3. Drag the Cube Prefab into the **Cube** field in the Inspector.
4. Adjust intensity and colors to your liking.

### Requirements:
- Unity 2022.3 LTS or higher (tested on Unity 6).
- URP/HDRP/Built-in (works everywhere).
